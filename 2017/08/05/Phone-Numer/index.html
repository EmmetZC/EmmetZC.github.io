我的新手机号

发表于 2017-08-05 分类于 Learning to Write 阅读次数：

临近毕业的时候，考虑到之后要到杭州工作，便新办了一个手机号。我并非不知道换手机的麻烦，各种业务绑定啦、新号码要通知亲朋好友啦等等。但是之前在学校用的套餐是本地资费，月租虽然不贵但是流量太少，而且校园套餐都是省内流量，想要带到杭州继续用的话实在不太划算。权衡之下还是选择了新办一个手机号。

很早以前就换过一次手机号，那次更换是因为移动的2.5G小水管实在无法满足日益增长的移动网络需求，所以顶着各方麻烦换了个联通号。犹记得那次换号工作持续了半个月，我挨个统计我有哪些地方用了旧号码，然后挨个将号码换绑。尽管已经尽可能做的漂亮，最后还是有几个漏网之鱼，不过总体而言还算比较好地完成了换号工作，除了2个月后我花了很久才把用旧号验证绑定的Facebook账号申诉回来。

有了上次的经验，加上有用新手机从老妈手上换来的旧手机作为备用机，我不需要太着急把旧号码注销，所以除了银行和支付宝我换成了新手机号，其他业务绑定的号码依旧是在学校用的旧号码。而且，最最关键的是，除了父母，我只把新手机号告诉2个朋友。这样做一方面是因为反正旧号码还是我的，虽然并没有随身携带，但是想要联系还是能联系上我；另一方面，我不知道怎么通知别人我换号了。从前我们换号以后会群发一条短信给所有联系人：“我是XXX，旧号码XXX因为某某原因不用了，这是我的新号码望惠存”，但是iPhone不让我一次性把我要的群发联系人选完，必须要一个一个点，虽然我没多少联系人但是这样实在费事；或者和现在很多人的做法一样，在朋友圈发一条“在XXX的新号码：XXX，望惠存”之类的，尽管实用，但是颇有一种张贴告示的感觉，个人不是很喜欢。所以想来想去，就不告诉大家了吧。

不过懒归懒，还是要保证别人用旧手机号找我的时候能找到我，所以我在旧号码的手机上装了IFTTT来监控来电和短信，这样只要将旧手机放在家里插着电连着Wi-Fi就可以将这些信息实时发送到主力机上。那么，在我部署好这个“转发”系统以后，借助这个系统接到过多少个别人的电话或者短信呢？

一个都没有。大家有事都上微信。
