Ep1-Sonnie's Edge

发表于 2019-07-21 分类于 Learning to Write 阅读次数：

- Aren't you scared?
- Scared? All the time.

桑尼的优势讲述了一个发生在科技高度发达的世界里的故事。这个世界里有种被称为“驯兽师”的职业，他们使用电子芯片将自己与巨兽的思维相连，进而完全控制巨兽进行搏斗——桑尼便是其中的佼佼者，她与她的巨兽四处参加地下搏斗比赛且未尝一败。某次比赛前黑道老板尝试花重金请她打假赛，却被她严辞拒绝。她的同伴解释道桑尼曾有着不堪回忆的过去，这份恨意让她不愿意输给任何人。在自己的驯兽师被桑尼击败以后，这位黑道老板设计探寻桑尼真正致胜的关键，却发现桑尼的优势远不在于“憎恨”这么简单。

与系列剧名 Love, Death&Robots 一样，第一集桑尼的优势全篇都散发着浓厚的赛博朋克的氛围。远处华丽的高科技大厦与近处低矮昏暗的厂房、以假乱真的全息影像与老旧的电控阀门都形成着鲜明的对比，霓虹灯带、荧光纹身和服饰也都是典型的赛博朋克元素。影片的核心虽然不在于巨兽，但是两个巨兽外形设计都很符合其驯兽师的特点，而且搏斗的场面制作也足够出彩，甚至桑尼的巨兽的战斗姿态如刺客一般迅捷而优雅，成为本片的一个小亮点。影片最后女子与桑尼短暂温存之后突然从指间伸出钢爪刺穿桑尼的头颅，这突如其来的感官冲击异常强烈，同时伴随谜底的揭晓呼应标题。可以说影片给观众带来的暴力美学的体验非常完整，尺度大但并不算过分（当然小朋友还是不要看），对观众的情绪调动恰到好处。本片的画面表现足够优秀，剧情设计毫不拖泥带水，并且充分利用了赛博朋克的背景设计故事与悬念——这一点非常重要，有不少所谓的赛博朋克作品只是借用了艺术风格或者世界观（说不好听点叫换皮），但是故事内核却与电影世界并没有太大的联系，典型例子就是 18 年上映的头号玩家。

总的来说，我本人很喜欢这一集，也挑不出什么毛病。不过毕竟是第一集，为了避免后面有更好的剧，所以这里不给满分，委屈一下给 4 星半。
