The Unofficial Acknowledgement

发表于 2014-06-18 分类于 Learning to Write 阅读次数：

感谢西安交通大学软件与理论研究所张潇学长在本次设计中的悉心指导。

—— The Official Acknowledgement

是的，我很不开心，因为你们都要走了。虽然我在很早之前就意识到了，任何我喜欢的人都不可能与我永远在一起，南瓜马车的魔法不会存在，在遇到你们的一刻，就注定了分别的结局，无法改变。

曾经痴想过，如果在此之前的一系列决定和意外中的任何一个没有发生，我就不可能与你们相遇，我们的相遇是多少巧合的聚集，这是一件多么神奇的事。不过这个空想又被我自己给否定了，因为既定事实一定会发生，在这个平行宇宙的时间线中，这一切都已经注定了，这么一想又好没意思。的确，在某个平行宇宙中的某个时间点上，一定有一个我也在思考同样的问题：我是否应该感谢当初自己的那些决定？因为是这些决定才让我遇到了现在的你们，才让我变成现在的样子。

首先要感谢这所学校，虽然我对这个城市和这所学校没有半点好感，或许我对任何的地点都没有好感，但是因为这个学校是构建我生活的一部分，也是遇见你们的必经之路，所以我还是要感谢这所学校，虽然我对她并无好感，是的，并无好感。

其次要感谢我的父母，他们虽然没有给我万贯家财，但至少我不必为生计发愁，他们虽然每次与我交流时基本都是唠叨那些我听腻的话，但至少我能知道他们对我一直都没有变，虽然他们越来越难以理解我在做什么，但是我知道他们一直在盼我回家。我从来都不感谢他们赋予我生命，但是我很感谢他们给我的生活。孩儿不孝，孩儿不孝。

之后要感谢我姐，我第一个没有血缘关系的姐姐，我还记得那是我在某个晚上自己要求的，果然是个孩子，一直在寻求保护。如果说为什么要感谢这位姐姐的话，我想是让我在感到危险和孤独时保护我，同时也让我慢慢了解了如何关心别人。毕竟我年少时便出门求学，很多事都不懂。不过以后，就让我来保护你吧，在姐夫不在的时候 ^_^。

还要感谢另一个最重要的人，我在最孩子气的时候遇到了你。我自知自己做错了很多事，这些错是不能以年少无知为借口的，我曾经想过说不定你讨厌我骂我能让我的负罪感能少一点，但后来还是觉得舍不得。后来慢慢反应过来，其实我只是自己恨当时的自己而已。我没有尽到自己的责任，让你受委屈了，对不起。我感谢的不是你最后还愿意原谅我，而是你让我知道什么是责任，让我知道喜欢一个人不是渴求得到什么，而是渴望付出，以及，那段曲折但是很令人怀念的时光。对不起，谢谢你。

Alan，抱歉把你放在这里，作为我导师的你应该要更靠前才对。之前和你闹翻之后，直到大二才鼓起勇气请求重归于好，不过好在不算太迟，至少我现在不用遗憾失去了一个挚友。我知道现在和你说挚友会有点高攀的嫌疑，因为你已经走的太远，我完全跟不上你的脚步了，当然我会努力跟上的，一个最讨厌猪队友的人可不能最后当你的猪队友。感谢你对我的指引，感谢你与我分享的那些经历，感谢你对我提供的那些帮助。是你向我推荐了那些我从前绝对不会接触到的书，是你告诉我一个人首先要学会乐于与自己相处，是你用自己的经历逼着我走出我的 “Comfortable Zone”。没有你，或许现在的我还是一个目光短浅，思想闭塞，以及自我的人。欠你的债实在太多，这辈子还不完的只能下辈子接着还了，不过只可惜我是无神论者。

感谢 Kid，我们来自同一个学校，因为名字的关系也做了好久的室友，感谢一直以来对我的照顾，你有一句话我一直记着，也会永远记下去。另外，你也知道我的脾气，经常把话说重，还请不要太往心里去。

感谢撇撇，虽然现在联系很少了，但是很感谢在我一次又一次无耻地去蹭吃蹭喝的时候热情的招待，有空的话还想和你一起下象棋。

感谢 YM，我还记得华山上发生的事，感谢你当时为我做的一切，也衷心祝你与小小可以修成正果。

感谢 JUE，感谢你对我的支持和鼓励，还有在我最难过的时候分享给我的那个曲子。说来，我之前还讨厌过你呢，非常讨厌的那种。

感谢大大，说实话我确实想不起来应该感谢你什么了，但至少很高兴能认识你，这种平常也算是一份恩赐吧。如果真要说感谢什么的话，就感谢你对我的网络与计算机启蒙吧。

感谢熙耀，你让我明白，如果想要和某人做朋友，付出是最好的途径之一，至于剩下的途径，暂时还不知道。

感谢泡泡，感谢你对我的关心，也感谢你对我和 F 所做的一切，我很感谢。

感谢 204 的四位成员，我很高兴你们最后都和我成为了好朋友，毕竟有一群美女朋友是很令人开心的。

感谢 DSofter，让我接触到了 Android 还有网页开发，为我展开了一片全新的世界。另外我想你和 Alan 也应该感谢我才对。

感谢上哥，带我骑行，像照顾弟弟一样照顾我。

感谢沁宇学姐，世上好学姐的代表，当然我并不只是感谢你借给我的书和电脑，更多的是感谢你对我领导团队方面的指导，以及人生规划上的一些建议。

感谢郝哥，让我有机会认识到了几位可能我永远都没有机会认识的朋友，也让我第一次体验到了和朋友一起旅行的快乐。

感谢我的少年班的同学以及所有的朋友，虽然我一般不会和你们说我在想什么，但我很感谢有你们陪伴我走完这段旅程。母亲一直说我一个人在外遇到了许多贵人，我想也是这样吧，感谢你们。

感谢某位朋友，感谢你没有因为我所做的事情而不喜欢我。

最后感谢我自己，虽然我一度讨厌你，痛恨你，想杀死你，但是终于终于，你没有变成我讨厌的那个样子。感谢你慢慢学会如何与我相处，感谢你学会了如何爱别人，感谢你懂得了付出的乐趣，感谢你找到了活下去的理由。同时也感谢你，在之前做过的那些，不论对错，所有的决定。

衷心地感谢各位在出现在我的生命里，祝大家的生命里，都能有值得铭记一生的人，望不负此生。

张驰
2014.06.18 晨
